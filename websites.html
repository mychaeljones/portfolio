PDX Guitar Duo (2011)

The PDX Guitar Duo asked me to create a website that they could easily update and maintain. I chose to create a website using WordPress, which is a blogging platform that has slowly turned into a content management system that is easy to use for non-technical users. This site uses some custom coding, including both JavaScript and PHP, as well as some custom MySQL database manipulation for handling performance dates.

Check out their site at www.PDXGuitarDuo.com.

Martians With Mustaches (2013)

Martians With Mustaches was an exhibit that was showcased by Washington State University Vancouver's Creative Media and Digital Culture program at the Nouspace Gallery in Vancouver, Washington. I was asked to create a website for the exhibit, which I quickly did. The site uses HTML5, CSS3, and jQuery.

More information on the Martians With Mustaches website and exhibit can be found at http://dtc-wsuv.org/martians/.

Transpoetica (2014)

Transpoetica was a digital poetry exhibit showcasing the work of Stephanie Strickland at Nouspace Gallery in Vancouver, Washington. I was asked to create a website for the exhibit, which was curated by the students of the Creative Media and Digital Culture program at Washington State University Vancouver. The website makes use HTML5, CSS3, jQuery, and PHP.

More information on Transpoetica please visit the website which can be found at http://dtc-wsuv.org/transpoetica/.
